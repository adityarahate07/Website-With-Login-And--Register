# 🔐 Login & Registration Website

## 📌 Project Description

This is a responsive **Login & Registration Web Page** built using **HTML, CSS, and JavaScript**.
It includes a modern UI with a popup form, smooth animations, and user-friendly design.

---

## 🚀 Features

* ✅ Login & Registration Form
* ✅ Popup Modal Design
* ✅ Smooth Animations
* ✅ Responsive Layout (Mobile Friendly)
* ✅ Clean UI using Flexbox
* ✅ Background Image Support

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Flexbox, Animations)
* JavaScript (DOM Manipulation)
